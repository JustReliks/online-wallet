Информация о типах счета
{{type.name}}

Накопительный счет - это счет, который предлагает не только хранения денежных средств, но и возможность снятия в любой момент.

Начисление средств: разрешено

Списание средств: разрешено

Создание цели: разрешено

Кредитный счет - это счет, который предназначается для учета кредитной задолженности перед банком по кредитному договору.

Начисление средств: разрешено

Списание средств: запрещено

Создание цели: запрещено

Зарплатный счет - это счет, который позволяет распоряжаться денежными средствами, полученными в качестве заработной платы.

Начисление средств: разрешено

Списание средств: разрешено

Создание цели: запрещено

Сберегательный счет - это счет, который предполагает сбережение средств на определенную цель

Начисление средств: разрешено

Списание средств: разрешено(после определенной даты)

Создание цели: разрешено
